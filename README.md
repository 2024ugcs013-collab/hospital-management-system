# Hospital Management System

Monorepo scaffold for a hospital management system with a React frontend and Node.js backend.

## Structure

- `frontend/` - Vite + React application
- `backend/` - Express API
- `docs/` - Project documentation

## Next Steps

1. Install dependencies in `frontend/` and `backend/`.
2. Wire environment variables in `.env` files.
3. Implement authentication, role-based routing, and API integration.
